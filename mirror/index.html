Welcome to apt.aris.web.id!

This service provides mirrors for the following archive repositories:

  • /ubuntu/
  • /debian/
  • /debian-security/

Supported distribution:

  • Debian Stretch
  • Ubuntu Bionic Beaver

To use it with a sufficiently recent apt, you for instance can put:

deb http://apt.aris.web.id/debian stretch main contrib non-free
deb http://apt.aris.web.id/debian stretch-updates main contrib non-free
deb http://apt.aris.web.id/debian-security stretch/updates main contrib non-free

or for Ubuntu:

deb http://apt.aris.web.id/ubuntu bionic main restricted universe multiverse
deb http://apt.aris.web.id/ubuntu bionic-updates main restricted universe multiverse
deb http://apt.aris.web.id/ubuntu bionic-security main restricted universe multiverse
deb http://apt.aris.web.id/ubuntu bionic-proposed main restricted universe multiverse
deb http://apt.aris.web.id/ubuntu bionic-backports main restricted universe multiverse

in your sources.list.

This service is also available on HTTPS,so with the apt-transport-https.
